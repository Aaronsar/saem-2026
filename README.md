# Salon des Études de Médecine 2026

Site officiel du salon — dimanche 8 novembre 2026 à Paris.

## Démarrer

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Accueil (hero, infos, CTA inscription)
- `/exposants` — Liste des exposants
- `/conferences` — Programme des conférences
- `/inscription` — Formulaire d'inscription

Les inscriptions sont enregistrées dans `data/inscriptions.jsonl` (non versionné).
